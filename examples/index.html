Simple Carousel JS

A jQuery plugin to create unstyled bare bone carousels with just the essential functionality available.

Why ?

Too many projects required carousels with custom looks and animations but same basic functionality e.g next, previous, dot navigation, play content animations during or after carousel animation is complete.

Dependencies ?

  • jQuery
  • Greensock's TweenMax

Whats available ?

  • basic carousel controls
  • optional dot pagination
  • check wether carousel is currently on the last slide or first
  • plenty of bindable events
  • accepts custom animations to override the default animation

What are the limitations ?

  • assumes all slides will be the same height

Basic Example

a simple carousel with images and dot navigation

HTML:

					
<div class="carousel">
	<div class="carousel_wrapper">
		<div class="carousel_slide">
			<img src="images/1.jpg">
		</div>
		<div class="carousel_slide">
			<img src="images/2.jpg">
		</div>
		<div class="carousel_slide">
			<img src="images/3.jpg">
		</div>
	</div>
</div>

CSS:

					
/* BEGIN: Basic Carousel */
.carousel .carousel_wrapper {
	position: relative;
	overflow: hidden;
}
.carousel .carousel_slide {
	position: absolute;
	left: 0;
	top: 0;
}
.carousel .carousel_slide img { width: 100%; }
/* END: Basic Carousel */

/* BEGIN: Dot Styles */
.carousel .carousel_pagination {
	position: absolute;
	bottom: 10px;
}
.carousel .carousel_pagination .dot,
.carousel .carousel_pagination .dot a {
	display: block;
	width: 10px;
	height: 10px;
}
.carousel .carousel_pagination > ul,
.carousel .carousel_pagination .dot { float: left; }
.carousel .carousel_pagination .dot { margin-right: 10px; }
.carousel .carousel_pagination .dot a {
	text-indent: -99999px;
	background: transparent url('../images/dot.png') no-repeat left top;
}
.carousel .carousel_pagination .dot.active a,
.carousel .carousel_pagination .dot a:hover  { background-position: left bottom; }
/* END: Dot Styles */

JS:

					
$('.carousel').simpleCarousel({ loop : true });

window.next = function () {
	$carousel.simpleCarousel('next');
};
window.prev = function () {
	$carousel.simpleCarousel('previous', { loop : false });
};

JS:

					
// custom animation
var custom = {
	'cross-fade' : function ($target, $old, o, next) {
		TweenMax.to($old, 1, { opacity : 0, ease : Power2.easeIn });
		TweenMax.set($target, {xPercent : 0, opacity : 0});
		TweenMax.to($target, 1, { opacity : 1, ease : Power2.easeIn, onComplete : function () {
			if (typeof next !== 'undefined') next();
		}});
	}
};
// init plugin
$carousel.simpleCarousel({ debug : true, animation : 'cross-fade', animations : custom });
